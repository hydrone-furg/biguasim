Welcome to BiguaSim’s documentation!

BiguaSim is a high-fidelity simulator developed by Nautec at Federal University of Rio Grande .

Features

  1. 3+ rich worlds with various infrastructure for generating data or testing underwater algorithms

  2. Complete with common underwater sensors including DVL, IMU, optical camera, various sonar, depth sensor, and more

  3. Highly and easily configurable sensors and missions

  4. Multi-agent missions, including optical and acoustic communications

  5. Novel sonar simulation framework for simulating imaging, profiling, sidescan, and echosounder sonars

  6. Imaging sonar implementation includes realistic noise modeling for small sim-2-real gap

  7. Easy installation and simple, OpenAI Gym-like Python interface

  8. High performance - simulation speeds of up to 2x real time are possible. Performance penalty only for what you need

  9. Run headless or watch your agents learn

  10. Linux and Windows support
