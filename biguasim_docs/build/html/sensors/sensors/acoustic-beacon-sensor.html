Acoustic Beacon

An accoustic modem for communications between underwater agents.

The command biguasim.environments.BiguaSimEnvironment.send_acoustic_message() is used to send messages between acoustic beacons. There’s a number of message types that can be sent, all with varying functionality. See biguasim.sensors.AcousticBeaconSensor for details.

Further, a few helper functions exist if needed:

  • biguasim.environments.BiguaSimEnvironment.beacons returns all beacons.

  • biguasim.environments.BiguaSimEnvironment.beacons_id returns all beacons’ ids.

  • biguasim.environments.BiguaSimEnvironment.beacons_status returns all beacons’ status (whether it’s transmitting or not).

See Multi-Agent Communications for an example of how to use this sensor.

See AcousticBeaconSensor for the Python API.

Example sensor definition:

{
    "sensor_type": "AcousticBeaconSensor",
    "location": [0,0,0],
    "configuration": {
        "id": 1
    }
}
